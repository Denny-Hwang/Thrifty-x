# Raspberry Pi 5 배포 검증 체크리스트 (RX Site)

작성일: 2026-05-06

## 1) 사전 준비
- [ ] Raspberry Pi OS 64-bit 설치 완료
- [ ] 시스템 업데이트 완료 (`apt update && apt upgrade`)
- [ ] Airspy 연결/전원/안테나 구성 완료
- [ ] 저장소 clone 및 Python venv 준비 완료

## 2) 설치 검증
- [ ] `libairspy` 설치 완료
- [ ] `pip install -e .[all]` 성공
- [ ] `python -c "import thriftyx"` 성공
- [ ] Airspy serial 열거 성공

## 3) 기능 검증
- [ ] `thriftyx capture ...` 60초 동작 성공
- [ ] `thriftyx detect ...` 산출물 생성 성공
- [ ] 산출물 파일 크기/개수 기대치 확인

## 4) 운영 검증
- [ ] systemd 서비스 등록/시작 성공
- [ ] 재부팅 후 자동기동 성공
- [ ] 장애(프로세스 kill) 후 자동 재시작 성공
- [ ] 로그 수집/조회(journalctl) 가능

## 5) 안정성 검증
- [ ] 2시간 연속 운용 성공
- [ ] 24시간 soak test 성공 (권장)
- [ ] 메모리 사용량 급증/누수 없음
- [ ] 샘플 드롭/유실률 허용치 이내

## 6) 저장소/복구 검증
- [ ] 디스크 80% 이상 시 경고 확인
- [ ] 디스크 90% 이상 시 정리 정책 동작
- [ ] 네트워크 단절/복구 후 파이프라인 회복

## 7) 판정
- [ ] Go (현장 배포 가능)
- [ ] Conditional Go (조건부 배포)
- [ ] No-Go (개선 후 재검증)
